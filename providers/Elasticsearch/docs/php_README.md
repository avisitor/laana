## Project Structure

The Hawaiian Search System consists of:

1. **PHP Backend** (`src/`) - Core search and indexing functionality
2. **Embedding Service** (`embedding_service/`) - Python-based vector embedding service
3. **Scripts** (`scripts/`) - Utility scripts for testing, maintenance, and administration
4. **Tests** (`tests/`) - Comprehensive test suite

## Getting Started

1. Review the test documentation in `tests_README.md` for system overview
2. Check the test suite summary in `TEST_SUITE_SUMMARY.md` for functionality details
3. See `embedding_service_requirements.txt` for Python dependencies
