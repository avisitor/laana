What is Noiʻiʻōlelo?

Noiʻiʻōlelo is a tool for learners of Hawaiian to search for sample sentences using a word or words and for those seeking to understand Hawaiian language patterns. It includes a database of sentences as well as a database of the full text of a large number of historical and current Hawaiian language documents. It is inspired by the Laana website of Zachary Silva.

Besides simple word searches, Noiʻiʻōlelo supports regular expression searches for complex patterns across a large number of historical and current documents. Our hope is that it will be useful for research into past and current Hawaiian grammar and lexical usage.

Search options

The default search option (selected from "Search Options") is "Any". All searches except for "Regex" and "Exact" are case-insensitive. These are the choices:

  • Any: matches sentences with any of the words in your search expression, e.g. hale kuai returns sentences containing hale, kuai or both
  • All: matches sentences that includes all the words in your search expression in any order, e.g. hale kuai returns sentences like Ua koha oia i ka hale no ke kuai ana as well as Ua kuai oia i ka hale
  • Exact: matches your search expression exactly as entered. Whether or not Hawaiian characters are treated the same as characters without diacritical marks depends on the No Diacriticals setting below. If No Diacriticals is not set, hale kuai returns sentences containing hale kuai but not hale kū‘ai while if it is set, both are returned
  • Regex: matches your regular expression, e.g. ho\w{5}\skaua to find sentences containing a 7-letter word starting with ho and followed by a space and kaua
  • No Diacriticals: applies only to Regex and Exact searches and means diacritical marks and ‘okina are ignored. For example, searching for ho‘okipa with Regex and No Diacriticals matches hookipa as well as ho‘okipa; searching for hookipa returns the same results. Without No Diacriticals, searching for ho‘okipa returns only results with ho‘okipa while searching for hookipa matches only on hookipa

The default sort option (selected from "Search Options") is "Random". Search results are returned in random order. These are the choices:

  • Random: returns search results in random order
  • Alphabetical: returns search results in alphabetical order
  • By source: returns search results by source document and then in alphabetical order
  • By length: returns search results by sentence length in ascending order
  • By length descending: returns search results by sentence length in descending order

Where do the sentences come from?

  • Ulukau (all Hawaiian-language documents)
  • Nupepa (all newspaper issues that have been transcribed)
  • Ke Aolama (all articles to )
  • Ka Ulana Pilina (Civil Beat, all articles to )
  • Kauakukalahale (Honolulu Star Advertiser, all articles to )
  • Ka Paa Moolelo (all documents)
  • Baibala (full bible, 2012 edition) (New and Old Testament)
  • Ehooululahui (a few full moʻolelo from UHMC)
  • Various other individual documents
